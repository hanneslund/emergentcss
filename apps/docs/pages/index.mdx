# What is this?

A compiler that generates CSS. Inspired by Tailwind. Both the compiler and the documentation are wip. Experimenting with having a build step for CSS utils, where the syntax doesn't have to be selectors.

Basic syntax

```
@mediaquery:pseudoclass:util=value
```

## Inspiration
